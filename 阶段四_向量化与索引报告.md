## 二、核心技术决策

### 2.1 从 bge-zh 换到 bge-en

| 维度 | bge-small-zh(原) | bge-small-en-v2(新) |
|------|-------------------|---------------------|
| 训练语料 | 中文为主 | 英文为主 |
| 维度 | 512 | **384** |
| 适用 | 中文文献 | **英文文献** ✓ |
| 英文 PubMed 召回 | 弱 | **强** ✓ |

**数据是 PubMed 英文文献**,用 bge-en 显著更对。Q1 测试结果对比:
- bge-zh 时代:PMC212319 排第 1,后面混进其他文章
- **bge-en 时代:PMC212319 在 top-5 全占** ← 召回纯净度大幅提升

### 2.2 BGE Query Instruction(关键!)

**BGE 官方建议**:查询文本前加 instruction,提升检索效果。

```python
instructed_query = "Represent this sentence for searching relevant passages: " + user_query
```

- **文档嵌入**:不加 instruction(纯文本)
- **查询嵌入**:加 instruction
- 官方文档:[https://huggingface.co/BAAI/bge-small-en-v1.5](https://huggingface.co/BAAI/bge-small-en-v1.5)

**实测效果**:这是 bge-en 召回比 bge-zh 干净的重要原因之一。

### 2.3 ChromaDB Where Filter 限制(踩坑记录)

**支持的操作符**:`$eq, $ne, $gt, $gte, $lt, $lte, $in, $nin, $and, $or`
**不支持**:`$contains, $regex, $like` 等字符串子串匹配

**重要约束**:**每个 where 字典只能有一个 operator**,多条件必须用 `$and` 包:

```python
# ❌ 错误
where_filter={"chunk_index": {"$gte": 0, "$lte": 5}}   # 报错!

# ✅ 正确
where_filter={"$and": [
    {"chunk_index": {"$gte": 0}},
    {"chunk_index": {"$lte": 5}},
]}
```

**应对子串匹配**(如"title 含 ARF"):
- 应用层过滤:召回 top-N 后 substring 匹配
- 预提取 tags:建索引时提取关键词进 metadata 数组,用 `$in` 匹配
- 换 ES / Milvus(支持全文检索)

## 六、关键技术总结

### 6.1 嵌入模型

```python
embedder = SentenceTransformer("BAAI/bge-small-en-v1.5")
embeddings = model.encode(texts, batch_size=64, normalize_embeddings=True)
```

**关键参数**:
- `normalize_embeddings=True`:bge 官方建议,余弦距离 = L2 距离时
- `batch_size=64`:平衡显存和速度
- `device=cuda`:如果可用就用 GPU

### 6.2 ChromaDB 集合

```python
client = chromadb.PersistentClient(path="./vector_db")
collection = client.create_collection(
    name="medical_papers_v4",
    metadata={"hnsw:space": "cosine"},
)
```

### 6.3 批量写入

```python
for i in range(0, len(texts), 1000):
    collection.add(
        ids=ids[i:i+1000],
        embeddings=embeddings[i:i+1000].tolist(),
        documents=texts[i:i+1000],
        metadatas=metadatas[i:i+1000],
    )
```

### 6.4 Query(支持元数据过滤)

```python
# 基础
results = collection.query(query_embeddings=[query_emb], n_results=5)

# 元数据过滤
results = collection.query(
    query_embeddings=[query_emb],
    n_results=5,
    where={"doc_id": "12969509"},                    # $eq
    # where={"doc_id": {"$in": ["12969509", ...]}}, # $in
    # where={"$and": [...]},                          # $and
)
```

### 6.5 BGE Query Instruction

```python
# 文档:不加
model.encode([doc_text])

# 查询:加
instructed = "Represent this sentence for searching relevant passages: " + query
model.encode([instructed])
```
